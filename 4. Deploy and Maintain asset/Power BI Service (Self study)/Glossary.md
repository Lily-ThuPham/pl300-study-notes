## Glossary of Power BI Service Components

| Term                   | Definition / Purpose                                                                                                                                                          | Key Characteristics                                                                                                                                                                                                                                                        |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Power BI service**   | The Software as a Service (SaaS) part of Power BI, accessed through a web browser, used to consume, interact with, and share reports and other content with a wider audience. | <ul><li>Accessed via `app.powerbi.com`.</li><li>Used for collaboration and distribution after building reports in Power BI Desktop.</li></ul>                                                                                                                              |
| **Workspaces**         | Collaborative spaces used to organize and manage collections of Power BI content.                                                                                             | <ul><li>Contains reports, semantic models, and dashboards.</li><li>Can be created for different teams or projects.</li><li>Enables collaboration and sharing.</li></ul>                                                                                                    |
| **Reports**            | An interactive, multi-page collection of visuals that provide a deep dive into a specific dataset.                                                                            | <ul><li>Created in Power BI Desktop and published to the service.</li><li>Can have one or more pages with multiple interactive visuals.</li><li>Based on a single semantic model.</li></ul>                                                                                |
| **Semantic models**    | A source of data that is ready for reporting and visualization. It represents the data model.                                                                                 | <ul><li>Consists of tables, relationships, and measures.</li><li>Can be reused to create many different reports.</li><li>Can be **promoted** or **certified** to help users find trustworthy data.</li></ul>                                                               |
| **Dashboards**         | A single-page, visual summary that provides a high-level overview of key metrics.                                                                                             | <ul><li>Exclusive to the Power BI service (cannot be created in Desktop).</li><li>Composed of **tiles** pinned from one or more reports.</li><li>Can support **data alerts** on certain tiles.</li><li>Focuses on quick insights; less interactive than reports.</li></ul> |
| **Apps**               | A way for designers to bundle and share related content (dashboards, reports, semantic models) with a broad audience.                                                         | <ul><li>Organizes related content into a single package for easy discovery.</li><li>Gives the designer control over content and user access.</li><li>Allows version control for releasing updates.</li></ul>                                                               |
| **Power BI gateway**   | A bridge connecting Power BI Service with on-premises data source                                                                                                             | <ul><li>Ensure security connection and retrieving of data.</li><li>2 types: Personal mode & Personal mode</li><li>A software which need to be installed on the computer</li>                                                                                               |
| **Workspace OneDrive** | Workspace OneDrive is a feature which allow configuring a Microsoft 365 Group whose SharePoint library is available to workspace users.                                       | A best practice is to give¬†[access to the workspace](https://learn.microsoft.com/en-us/power-bi/collaborate-share/service-give-access-new-workspaces)¬†to the same Microsoft 365 Group whose file storage you configured.                                                   |
|                        |                                                                                                                                                                               |                                                                                                                                                                                                                                                                            |
